## About the Project

FOODO is a modern restaurant website built with HTML and CSS. It presents a clean and responsive layout that highlights menu items, special offers, business hours, a food gallery, and contact information in an organized and visually appealing way.

The project focuses on creating an attractive landing page experience while maintaining a simple and easy-to-navigate design.



## Features

- Responsive layout for different screen sizes
- Hero section with a background image and call-to-action button
- About section introducing the restaurant
- Special offers section
- Menu with featured food items and prices
- Business hours display
- Image gallery
- Contact section with a message form
- Smooth scrolling navigation
- Hover effects and modern styling



## Technologies Used

- HTML5
- CSS3
- Google Fonts (Quicksand)
- Font Awesome
